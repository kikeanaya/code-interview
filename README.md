# Phone Catalogue
This app shows the user a list of phones and when they click on one model it shows them more details for that phone.

## Setup

To run this project you have two options:

### With Docker Compose

Go to the root folder of the project and run the following commands:

```
cd exercise
docker-compose build
docker-compose up
```

Go to http://localhost:3000/ on your browser.

### Without Docker Compose

Install it locally using npm:

To run the API, go to the root folder of the project and run the following commands:

```
cd server
npm install
npm start
```

To run the React app, go to the root folder of the project and run the following commands:

```
cd client
npm install
npm start
```

Go to http://localhost:3000/ on your browser.
